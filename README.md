This organization brings together students from Wrocław University of Science and Technology to develop a project focused on Information Systems Integration. The main goal is to design and implement a microservices-based system — each microservice has its own dedicated repository. Additionally, there is a separate repository for project documentation and supporting materials.

**Key Points:**
- Each microservice is implemented as a standalone repository within this organization
- The documentation repository contains comprehensive guides, system architecture, and integration details
- The project encourages teamwork, best coding practices, and hands-on experience in distributed systems and microservices architecture
